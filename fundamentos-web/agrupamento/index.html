introdução

A linguagem HTML tem papel importante no desenvolvimento de sites, portais, e-commerce, aplicativos web e ate APPs

Isso pois, o HTML permite criar paginas semanticamente formatados trazendo o conteúdo organizado.

A linguaguem possui diversas TAGs de marcação, por exemplo as principais são:
<HTML>
Informa que o documento é um HTML
<HEAD>
Trás o cabeçalho do arquivo HTML
<BODY>
Onde consta o conteúdo geral de uma pagina
