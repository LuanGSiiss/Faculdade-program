Neste projeto, foi desenvolvido a modelagem de um banco de dados para o restaurante Madero, focado em seu cardápio. 
Utilizando o software SQL Power Architect, foi criado a estrutura das tabelas. 
Após a modelagem, foi realizado a criação das tabelas, trigger, views e inserção dos dados.
Todo SQL foi criado manualmente e implantado no banco de dados PostgreSQL.
